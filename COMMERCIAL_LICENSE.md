# Bots Nest 商业使用授权

## 概述

Bots Nest 在 [AGPL v3](./LICENSE) 下开源发布。AGPL v3 是一个强 copyleft 协议，要求：

- 任何修改后的版本必须公开源代码
- 通过网络提供服务时也必须提供源代码
- 衍生作品必须以相同协议发布

对于不希望公开自身代码修改的企业用户，我们提供**商业使用授权**，允许在保留代码私有的前提下合法使用、修改和部署 Bots Nest。

## 商业授权 vs AGPL v3

| | AGPL v3（开源版） | 商业授权 |
|---|---|---|
| 使用方式 | 免费 | 付费 |
| 修改代码 | 必须公开修改 | 无需公开 |
| 内部部署 | 必须提供源码获取方式 | 完全私有 |
| 商业化运营 | 需开源衍生作品 | 不受限制 |
| 技术支持 | 社区（GitHub Issues） | 优先技术支持 |
| 法律保障 | 无 | 授权书 + 合规保障 |
